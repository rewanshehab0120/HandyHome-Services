## 📋 Project Overview
This project involves the development of a comprehensive **Business Intelligence Dashboard** for **HandyHome Services**. The primary objective was to transform raw operational and marketing data into an interactive visual story that facilitates data-driven decision-making for stakeholders.

The project highlights the intersection of **Data Analytics** and **UX Design**, ensuring that complex insights are accessible and actionable.

## 🚀 Key Features & Technical Implementation

### 1. Interactive Navigation & UX
* **Centralized Landing Page:** Developed a custom navigation system using **bookmarks and buttons** to allow seamless switching between Marketing, Customers, and Operations segments.
* **Smart Hover Tooltips:** Implemented report page tooltips that display brief descriptions and KPIs when hovering over navigation elements, enhancing the user experience.
* **Consistent Branding:** Meticulously aligned the dashboard's theme and color palette with the company's official branding for a professional finish.

### 2. Marketing Analytics
* **ROAS & CPA Tracking:** Created advanced **DAX measures** to calculate Return on Ad Spend (ROAS) and Cost Per Acquisition (CPA).
* **Channel Performance:** Identified Instagram as the most efficient growth driver with a ROAS of **~6.0**.

### 3. Operational & Customer Insights
* **Retention Modeling:** Built a model to analyze the balance between new and returning customers, revealing a **50% retention rate**.
* **Peak Time Analysis:** Used **Power Query (M Language)** to categorize shifts and identify peak operational hours (Afternoon shift at 26.7%) for better resource allocation.

## 🛠️ Tech Stack
* **Power BI Desktop:** For data modeling and visualization.
* **DAX (Data Analysis Expressions):** For advanced calculated measures and business logic.
* **Power Query (M Language):** For ETL processes and data cleaning.
* **Figma/Canvas:** Used for background design and layout planning.
